# Projet de statistique sur R - Facteurs influençant le bonheur

Projet réalisé en 2023 par BONNET Kloé, DONAT-BOUILLUD Lallie et GROUILLER Martin.

## Introduction
Dans ce projet, nous avons décidé de nous intéresser au bonheur et particulièrement aux facteurs qui peuvent contribuer ou détériorer le bonheur ressenti. 
Ainsi, avec les outils statistiques dont nous disposons et grâce à des études sur un grand nombre d'individus, nous espérons pouvoir dégager des tendances 
pour connaître les facteurs qui peuvent influer sur le bonheur des individus. Le rapport mondial du bonheur est un rapport publié annuellement par les Etats Unis depuis 2012. 
Ce rapport a pour objectif d’essayer de quantifier le bonheur des différentes populations géographiques et de les comparer avec d’autres critères. 
Nous avons décidé d’étudier les données de 2020 publiées sur le site DASL, afin d’étudier comment le bonheur varie selon les secteurs géographiques et de voir 
comment les différents facteurs peuvent avoir un impact les uns sur les autres et influer le bonheur ressenti ou non d’une population.
